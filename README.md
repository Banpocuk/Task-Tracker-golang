# Task-Tracker-golang
Task Tracker made by Banpocuk. Language - Golang. Actual commands in READ.ME

commands - description

help - see commands
exit - exit programm
write - add task in .txt file
list - list all task
list by id - list all tasks by  id
list by status - list all tasks by some status
update by id - updating status/task. Finding task/status by id
update by status - updating status/task. Finding task/status by status
delete - simply deleting task. Programm find them by id/status

Simply write some commands and there will be intsruction on it.
